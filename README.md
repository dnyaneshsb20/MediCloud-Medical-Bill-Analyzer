**MediCloud â€“ Medical Billing Analyzer** is an interactive Streamlit-based healthcare analytics dashboard designed to analyze patient billing data, doctor performance, and revenue patterns.

Earlier, hospitals, clinics, and diagnostic centers manually recorded billing details, which made analysis slow, inconsistent, and prone to errors.

With this software:
- Billing data is analyzed automatically
- Doctors' revenue and performance become easy to compare
- Common symptoms and patient demographics can be tracked
- Insights like total revenue and avg billing are calculated instantly
- Filtered data can be downloaded as CSV for further use 

---

## Key Features
- Interactive dashboard with real-time medical billing insights
- Filters for Doctor, Gender, and Symptoms
- Shows total patients, total revenue, and average bill
- Doctor-wise revenue visualization
- Common symptoms and gender distribution charts
- Auto-generated dataset using Faker
- Download filtered data as CSV
- Lightweight and easy to use
